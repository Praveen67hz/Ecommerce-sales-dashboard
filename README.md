# E-commerce Sales Dashboard | Power BI

Developed an interactive Power BI dashboard to analyze sales performance and business KPIs for an e-commerce dataset.

## Project Objective
The objective of this project was to analyze sales, profit, quantity sold, customer behavior, and payment trends to derive actionable business insights.

## Key Metrics
- Total Sales: 438K
- Total Profit: 37K
- Quantity Sold: 5615
- Average Order Value: 121K

## Dashboard Features
- State-wise sales analysis
- Category-wise quantity sold
- Customer-wise purchase trends
- Payment mode distribution
- Monthly profit analysis
- Sub-category profit comparison

## Tools Used
- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization

## Business Insights
- Clothing contributed the highest share of quantity sold
- COD was one of the major payment methods
- Certain months showed negative profit trends
- Maharashtra and MP had strong sales contribution
